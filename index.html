☁️ Configurar Sincronização em Tempo Real

Use o Supabase (gratuito) para sincronizar entre todos os computadores.
supabase.com → Criar conta → Novo projeto → Copiar URL e chave anon.

URL do projeto (ex: https://xyzxyz.supabase.co)
Chave anon/public
⚡ SQL para criar a tabela (copie e cole no Supabase → SQL Editor):
create table if not exists cdgam_data (
  id text primary key,
  payload text,
  updated_at text
); 📋 copiar
Passo a passo:
1. supabase.com → login → seu projeto
2. Menu lateral → SQL Editor → cole o SQL acima → Run
3. Vá em Settings → API → copie Project URL e anon public key
4. Cole os dois campos acima e clique Salvar e Conectar

🔥 Configurar Salvamento na Nuvem

Seus dados já estão salvos localmente neste computador.
Para garantir que nunca perca nada mesmo desligando o PC, siga os passos abaixo para ativar o Firebase (gratuito, da Google).

PASSO 1 — Acessar o Console Firebase
1. Acesse console.firebase.google.com
2. Faça login com sua conta Google
3. Clique em GAM-SJP (projeto já criado)
PASSO 2 — Abrir Realtime Database
1. No menu lateral esquerdo clique em Build
2. Clique em Realtime Database
3. Se não existir ainda, clique "Create Database", escolha São Paulo e modo Test Mode
PASSO 3 — Liberar as Regras de Acesso
1. Clique na aba Rules
2. Apague o conteúdo e cole exatamente isso:
{"rules":{".read":true,".write":true}} 📋 clique para copiar
3. Clique em Publish
PASSO 4 — Testar a Conexão
1. Volte aqui no dashboard
2. No canto superior direito deve aparecer ☁ Nuvem ativa ✓
3. Salve qualquer dado — ele aparece no Firebase em tempo real!
💡 Segurança: As regras acima são abertas (ideal para rede interna do CD). Se quiser restringir o acesso, fale com o TI para configurar autenticação.
