☁️ Configurar Sincronização em Tempo Real

Use o Supabase (gratuito) para sincronizar entre todos os computadores.
supabase.com → Criar conta → Novo projeto → Copiar URL e chave anon.

URL do projeto (ex: https://xyzxyz.supabase.co)
Chave anon/public
Passo a passo:
1. Acesse supabase.com e crie uma conta gratuita
2. Crie um novo projeto (nome: cdgam, qualquer senha)
3. Vá em Settings → API e copie a Project URL e a anon public key
4. Vá em Table Editor → New Table, nome: cdgam_data
5. Adicione as colunas: id (text, primary key), payload (text), updated_at (text)
6. Cole a URL e a chave aqui e clique Salvar
7. Faça isso em todos os computadores com as mesmas credenciais
